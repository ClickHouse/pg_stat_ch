and hosted at [TBD](TO DO)

## Running locally

Run the following command to install the CLI:

<CodeGroup>
  ```bash npm theme={null}
  npm i -g mint
  ```

  ```bash pnpm theme={null}
  pnpm add -g mint
  ```
</CodeGroup>

Start the development server from this directory:

```bash
mint dev
```

The site will be available at `http://localhost:3000`.
